# Blog Spider

## docker images
* mongo
* redis
* python
* nginx

## init

### spider
 * cd spider/
 * docker-compose build
 * docker-compose exec mongodb mongo /script/init_admin.js
 * docker-compose exec mongodb mongo /script/init_db.js

### www
 * cd www/
 * docker-compose build

## run

### spider
 * cd spider/
 * docker-compose up

## www
 * cd www/
 * docker-compose up
