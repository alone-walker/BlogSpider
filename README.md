# Blog Spider

## IMAGES
* mongo
* redis
* python
* nginx

## INIT
### spider
 * cd spider/
 * docker-compose build

### www
 * cd www/
 * docker-compose build

## RUN
### spider
 * cd spider/
 * docker-compose up -d

## www
 * cd www/
 * docker-compose up -d
